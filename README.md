## 시흥시 내의 공공자전거 서비스 활성화에 기여하고 싶어 개발한 어플리케이션입니다!

## 제공 기능

 - 시흥시 내 공공대여소 위치, 정보 표시
 - 시흥시 추천 자전거 경로 표시
 - 출발지, 목적지 설정하여 자전거 경로 표시
 - 라이딩 시작 시 현재 속도와 라이딩한 시간 표시
 - 라이딩 종료시 기록 저장
